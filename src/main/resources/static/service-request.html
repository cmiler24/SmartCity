Help us keep the city clean and safe. Report potholes, broken streetlights, graffiti, and other service issues.

Reporting Tips

  • Be specific: Include as many details as possible about the issue location and nature
  • Include landmarks: Mention nearby stores or intersections if exact address is unclear
  • Safety first: If the issue poses an immediate danger, please call 911
